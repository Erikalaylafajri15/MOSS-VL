No, the application runs locally on your machine. You do not need the internet to process images.

Can I analyze multiple images at once?
The current version supports one image at a time. The software processes the current image until you select a new file.

Why does the analysis take time?
The software performs complex math for every image. A faster graphics card reduces the wait time.

What if the software crashes?
